- pertama buka di functions.php -> buat function dengan nama aqso_menus (dikasih "aqso" biar gak nabrak sama function wordpress lain) -> di functionnya buat variable array $locations yang dikasih parameter objek yang kiri adalah nama label dan yang kanan adalah deskripsi ("contoh => "deskripsi dari label") -> lalu dibawahnya panggil function wordpress yaitu register_nav_menus($locations) dan parameternya adalah array yang sebelumnya kita udah bikin -> lalu buat hook untuk manggil function kita -> di bawah functionnya tulis add_action("init", "aqso_menus") parameter pertama adalah init untuk nyruh function dipanggil pertama kali, parameter kedua adalah nama function yang mau kita jalanin.

- Lalu ke backoffice ke pages, pastiin ada beberapa halaman yang udah dibuat -> lalu ke appearance lalu pilih menu -> create new menu -> masukin nama contoh main menu -> lalu pilih di kiri halaman mana aja yang mau dimasukin lalu klik save -> dan ceklis lokasi penayangan navbar utama di atas (sesuai deskripsi di function)

- kalo udah ke header.php (tempat di mana navbar itu ditaro) lalu di tempat naro navbar-nya buka syntax php -> lalu ketik wp_nav_menu untuk manggil data dari menus -> lalu di dalem parameternya berupa array() di dalem array parameter dikasih "menu" => "primary", "container" => "", "theme_location" => "primary". menu untuk kasih nama misal primary, "container" sebagai pembungkus misal 'div', dan "theme_location" untuk pilih location yang udah kita bikin dan pilih sebelumnya.
